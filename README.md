#  Netflix Clone

A responsive **Netflix Landing Page Clone** built using **HTML5** and **CSS3**. This project recreates the look and feel of Netflix's homepage while helping improve front-end development skills.

## 📌 Features

* 🎥 Netflix-inspired UI
* 📱 Fully responsive design
* 🎨 Modern and clean interface
* 🖼️ Hero banner with background image
* 📺 Movie and TV show sections
* 🔴 Navigation bar
* 📧 Email subscription section
* ❓ FAQ section
* 👣 Footer with useful links

## 🛠️ Technologies Used

* HTML5
* CSS3

## 📸 Preview

This project is a front-end clone of the Netflix landing page created for educational and practice purposes.
